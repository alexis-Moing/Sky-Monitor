# SkyMonitor

Aplicacion Web para monitorear y rastrear estado de un dron el cual primero funciona
recibiendo datos de un script de python, pero posteriormente funcionara para analizar
la trayectoria de un dron real en vuelo.

## Stack 

Frontend: React.Js, Tailwindcss
Backend: Flask, PostgreSQL

## Flujo del Proyecto

La cabina (Frontend) -- Torre de control (Backend) -- Caja negra (Base de datos)

- Frontend: React
- Backend: FLask
- Base de datos: Postgresql
